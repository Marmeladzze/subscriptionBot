# subscriptionBot
Навайбкоденный бот для  функции платной подписки у канала  
Как его запустить:  
Скачиваете папку с файлами  
Скачиваете зависимости для работы файлов: pip install -r requirements.txt  
Создаете файл .env  в этой же папке(БЕЗ РАСШИРЕНИЯ, ПРОСТО .env)  
Копируете 3 строки снизу и вставляете в .env:  

BOT_TOKEN="Токен вашего бота(получаете у botFather)"  
ADMIN_IDS="Айди вашего аккаунта телеграмм"  
PAYMENT_PROVIDER_TOKEN="Токен вашего платежного провайдера(получаете у botFather)"  

ВАШ БОТ ДОЛЖЕН БЫТЬ АДМИНОМ ВАШЕГО ТЕЛЕГРАММ КАНАЛА  

После заполнения данных в .env, пишете в терминал python main.py и бот должен запуститься!  
Поздравляем, у вас есть готовый бот!  

По поводу функций бота:  

Для обычного пользователя:  
Оплата (после оплаты выдается одноразовая временная ссылка пользователю)  
Информация(Заполняется админом)  
Профиль(Показывает статус подписки и ее длительность)  
Поддержка(Отправляет админу сообщение)  

Для админа: (Чтобы вызвать админ-панель, пропишите /admin (Чтобы сработало, в файле .env должен быть ваш айди))

Рассылка  
Статистика (кол-во пользователей, выручка и самый частый тариф)  
Промокоды  
Управление тарифами  
Пользователи (можно по айди найти любого участника канала и удалить/добавить подписку)  
Смена канала  
Изменение фото приветствия  
Добавление информации (22 строка)
